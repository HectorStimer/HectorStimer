##  Projetos em Destaque

### [Encurtador de Links (URL Shortener API)](https://github.com/HectorStimer/URL-Shortener)
API de alta performance para encurtamento de URLs, focada em resiliência e baixa latência.
- **Performance:** Implementação de cache de dois níveis (Redis + DB) com estratégia de fallback.
- **Segurança & Resiliência:** Proteção contra ataques de força bruta via **Rate Limiting (Bucket4j)** e autenticação JWT com Spring Security.
- **Arquitetura:** Versionamento de banco de dados com **Flyway** e documentação automática com **Swagger (OpenAPI)**.
- **Testes:** Cobertura de lógica de negócio com **JUnit 5 e Mockito**.

**Stack:** `Spring Boot`, `Redis`, `PostgreSQL`, `Flyway`, `Bucket4j`, `Docker`.

---

###  [Sistema de Gestão Hotdogtaz](https://github.com/HectorStimer/web-hotdogtaz-back)
Ecossistema completo para gestão de lanchonetes, controlando desde o estoque até o fechamento de comandas.
- **Core Business:** Implementação de regras complexas para fechamento de pedidos e recálculo dinâmico de totais.
- **Gestão de Identidade:** Integração com **Keycloak** para controle de acesso granular baseado em roles (Admin, Waiter, Cook).
- **Frontend:** Interface administrativa desenvolvida em **React + TypeScript**.

**Stack:** `Spring Boot`, `PostgreSQL`, `Keycloak`, `React`, `TypeScript`, `Docker`.

---

###  [Módulo de Atualização ERP (Auto-Updater)](https://github.com/HectorStimer/erp-launcher)
Solução de automação para distribuição de software legado.
- **Integração Cloud:** Consumo da **Google Drive API** para verificação de versões e download de patches.
- **Client-Side:** Interface desktop desenvolvida em **JavaFX** integrada ao core Spring.
