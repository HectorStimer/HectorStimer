##  Projetos em Destaque

###  [Encurtador de Links (URL Shortener API)](https://github.com/HectorStimer/URL-Shortener)
> API robusta focada em performance e baixa latência para redirecionamento massivo de URLs.

* **Resiliência:** Cache de dois níveis (**Redis + Postgres**) com fallback estratégico.
* **Segurança:** Proteção via **Rate Limiting (Bucket4j)** e filtros de segurança JWT.
* **Engenharia:** Documentação interativa com **Swagger** e migrações via **Flyway**.
* **Qualidade:** Cobertura de testes unitários com **JUnit 5 e Mockito**.

---

###  [Sistema de Gestão Hotdogtaz](https://github.com/HectorStimer/web-hotdogtaz-back)
> Ecossistema Fullstack para automação comercial de lanchonetes e restaurantes.

* **Negócio:** Regras complexas de fechamento de comandas e recálculo dinâmico de itens.
* **Identidade:** Gestão de acessos baseada em Roles via **Keycloak**.
* **Interface:** Dashboard administrativo responsivo com **React e TypeScript**.

---

###  [Módulo de Atualização ERP](https://github.com/HectorStimer/erp-launcher)
